Hello MySelf Aashu Singh , a Software and algorithms Enthusiast. Writing algorithms is more joyful than overthinking about it's complexity.


## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
